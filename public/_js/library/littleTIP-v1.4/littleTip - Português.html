littleTIP é um elemento comum de interface gráfica, utilizado para fornecer informações adicionais quando o usuário passa o cursor do rato sobre ele.

Pode ser usado num link ou num elemento span. Numa imagem como no exemplo em cima ou mesmo com uma imagem incorporada.

littleTIP pode usado com qualquer elemento html que tenha o atributo title.
